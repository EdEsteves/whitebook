## Available Scripts

### `npm install`

Instala as dependências para rodar o projeto.


### `npm start`

Roda o projeto\
Abra [http://localhost:3000](http://localhost:3000) para ver no browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.
